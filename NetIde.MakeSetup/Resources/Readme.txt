The NetIDE.sfx files is based on the 7zS2.sfx file from
http://sourceforge.net/projects/sevenzip/files/7-Zip/9.22/7z922_extra.7z.

The resources have been changd with http://www.angusj.com/resourcehacker/
to include a manifest which executes the installer as invoker (i.e. suppress
UAC), to replace the icon and to change the version information.
